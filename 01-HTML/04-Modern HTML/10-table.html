HTML vs HTML5

HTML HTML5
HTML is the older version of the markup language used for structuring and presenting content on the web. HTML5 is the latest version of HTML with new features and enhancements.
HTML does not have specific tags for video, audio, and canvas elements. HTML5 introduces new tags like `video`, `audio`, and `canvas` to handle multimedia and graphics directly in the browser.
HTML lacks support for offline web applications and local storage. HTML5 provides features like offline storage, web workers, and application cache, enabling developers to build offline web applications.
