Tab-Index

Tab index is an HTML attribute (tabindex) used to control the order in which elements receive focus when navigating using the keyboard.
It assigns a numerical value to an element to specify its position in the tabbing order.
The tabindex attribute can be set to positive integers (such as 1, 2, etc.), zero (0), or negative integers (-1).
Elements with a lower tabindex value are focused first when tabbing through the page, and elements with the same tabindex value are focused in the order they appear in the HTML source.
