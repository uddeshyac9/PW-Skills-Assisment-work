Html-Entities

HTML entities are special codes or character references used in HTML to represent characters that have special meaning
or cannot be easily typed directly on a keyboard. They are represented by an ampersand (&) followed by a specific code or name,
and ending with a semicolon (;).

Here are five commonly used HTML entities:

  1. <- Represents the less-than symbol (<).
  2. > - Represents the greater-than symbol (>).
  3. & - Represents the ampersand symbol (&).
  4. " - Represents the double quotation mark (").
  5. © - Represents the copyright symbol (Â©).

These HTML entities allow you to include special characters in your HTML document without causing conflicts with the HTML syntax.
