What is HTML

HTML (HyperText Markup Language) is the standard markup language used for creating web pages. It provides the structure and content of a webpage.

What is Semantic HTML

Semantic HTML refers to the practice of using HTML tags that convey meaning and provide context to the content they enclose. Semantic tags make the structure and purpose of the content more understandable for both humans and search engines.

Common Semantic Tags

  • <header>: Represents the introductory content or a container for the introductory content of a section or page.
  • <nav>: Represents a section of the document that contains navigation links.
  • <main>: Represents the main content of a document, excluding headers, footers, and navigation sections.
  • <section>: Represents a standalone section of content within a document.
  • <article>: Represents a self-contained composition within a document, such as a blog post or a news article.
  • <footer>: Represents the footer of a document or a section, typically containing information about the author, copyright, and related links.
